# ARC V13 Formal V34/V35 Update

本版調整：

## V34 財務對帳確認批次繳費日期修改

- 財務對帳確認批次明細左上方顯示：繳費日期、繳款人、仲介、扣款帳戶、帳戶名稱、帳號後五碼。
- 管理員 / 會計可在對帳完成前修改整個繳費批次的繳費日期。
- 支援西元、民國年、斜線、點號、純數字日期格式。
- 修改後同步更新 `payment_batches.payment_date` 與該批次所有 `arc_cases.payment_date`。
- 修改不會新增批次、不會重複扣款、不會影響帳戶餘額。
- 修改會寫入操作紀錄。

## V35 會計角色權限與財務對帳確認餘額修改

- 會計左側選單只顯示：總覽、財務對帳確認、財務查詢、匯出資料。
- 會計不再看到「仲介與扣款帳號」獨立頁。
- 財務對帳確認頁新增：目前帳戶餘額、修改後餘額、差額、餘額調整原因。
- 管理員 / 會計可修改本批次所選扣款帳戶餘額。
- 餘額修改只調整該扣款帳戶，不影響其他帳戶、不影響案件金額、不新增批次、不重複扣款。
- 餘額修改寫入 `account_transactions` 與 `audit_logs`。
- 財務查詢展開批次後可查看該批次相關帳戶餘額異動紀錄。

## Supabase SQL

本版不新增資料表，不需要新 SQL。前提是前面 V28 / V30 SQL 已執行。
