# ARC V13.45

本版修正：

- 傳真領件單下方「總件數」改為「總張數」，依每筆案件張數加總。
- 傳真領件紀錄新增本次總張數顯示與保存欄位。
- 案件查詢新增「修改」功能，所有登入者可修改案件資料。
- 案件查詢修改雇主、工人、團號、申請項目、金額、張數、仲介、承辦、收費日期、收件編號、外字五碼、舊卡、經手人後四碼、收據順序、領件日與備註後，會依案件 ID 同步連動繳費、財務與傳真領件資料。
- 修改金額或申請項目會同步更新財務批次明細與批次總金額，不會重複扣款。
