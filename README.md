# ARC V13 Formal

V37.4 更新：財務查詢批次資訊版面修正。

本版重點：
- 修正財務查詢批次資訊跑到右側、左側大片空白的問題。
- 批次編號、查看明細、對帳狀態、繳費日期、繳款人、仲介、扣款帳戶、批次金額等資訊改為同一整排橫向顯示，寬度不足時才自動換行。
- 展開「查看明細」後，明細表維持一列一筆資料，必要時使用表格橫向捲動，不壓縮欄位。
- 不修改 Supabase SQL，不影響搜尋、繳費、對帳、權限與財務資料。

部署方式同前版：上傳完整 zip 到 Codespaces 根目錄，解壓後執行 npm install / npm run build / git push。
# ARC V13 V37.3

本版修正財務對帳確認與財務查詢明細可修改欄位。

新增/調整：
- 財務對帳確認明細可修改：雇主、工人、團號、入境日、申請日、申請項目、項目金額、修正原因。
- 財務查詢展開明細可由管理員/會計修改上述欄位。
- 明細表新增「申請日」欄位。
- 修改申請項目或項目金額時，會同步更新 payment_batch_items 與批次總金額。
- 修改雇主、工人、團號、入境日、申請日會同步更新案件主資料。
- 所有明細資料修改都會寫入操作紀錄，修正紀錄欄會顯示最近明細異動摘要。

本版不需要新增 Supabase SQL。
