# ARC V13 Formal｜V32 團號必填與已領件功能

## 本版重點

- 居留案件登記的團號改為必填。
- 單筆案件登記、批次送件、現場申請都會檢查團號。
- 批次送件逐列檢查，有錯誤列時不會送出任何資料。
- 傳真/領件的「移民署傳真領件」明細新增「已領件」按鈕。
- 已領件會要求輸入領件日，支援西元與民國日期格式，最後統一為 `yyyy-mm-dd`。
- 已領件後會把案件從待傳真/領件與預計領件區移除，案件保留於案件查詢，狀態顯示為「已領件」。
- 案件查詢新增顯示收費日期、傳真日期、領件日。
- 已領件動作會寫入操作紀錄。

## V33 更新重點

- 會計角色左側選單只保留：總覽、財務對帳確認、財務查詢、匯出資料、仲介與扣款帳號。
- 會計角色嘗試進入未授權頁面時會提示「您沒有進入此功能的權限。」並導回可用頁面。
- 會計總覽改為財務重點資訊：待對帳批次、已對帳批次、今日 / 本月繳費批次與帳戶餘額。
- 財務對帳確認只顯示待對帳、項目金額錯誤待確認的批次；對帳完成後會從待處理清單移除。
- 財務查詢改為以「繳費批次」為主體顯示，可展開查看批次明細。
- 財務查詢搜尋支援批次編號、繳費日期、繳款人、仲介、扣款帳戶、雇主、工人、團號與申請項目，搜尋後仍以批次顯示。
- 仲介與扣款帳號頁補上帳號後五碼、是否啟用、是否預設帳戶；會計僅可查看，不可調整餘額。
