# ARC V13.50.1｜簽收單張數與財務備註補強版

此版接續 ARC V13.50，重點是補強簽收單列印與移入傳真/領件狀態顯示，並修正原包建置時出現的 TypeScript 錯誤。

## 完成項目

1. 簽收單 / 領件單
   - 「本承辦總領件數」已改為「本承辦總張數」。
   - 依承辦分組各自加總張數。
   - 張數空白預設 1 張。
   - 張數 0、負數或非數字會阻擋列印並提示。
   - 簽收單明細新增「承辦」欄位。

2. 傳真領件紀錄
   - 每筆批次紀錄只保留一個「列印簽收單」按鈕。
   - 點選後依該批次全部案件列印，不混入其他批次。

3. 居留證繳費
   - 待繳案件可點「移入傳真/領件」。
   - 不建立繳費批次、不扣款、不進財務對帳確認。
   - 成功後案件進入【傳真/領件】→【移民署傳真領件】。
   - 案件查詢顯示「不需繳費 / 待加入預計領件」。

4. 財務對帳完成備註
   - 對帳完成彈窗可輸入備註。
   - 備註非必填。
   - 對帳完成後備註會保存並顯示於財務查詢。
